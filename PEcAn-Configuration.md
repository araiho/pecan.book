h1. Pecan Configuration File
The PEcAn system is configured using a xml file, often called settings.xml. The configuration file can be split in X seperate pieces:
# PEcAn folders
# database access
# Model setup
# Run setup
# PFTs
# Meta Analysis
# Ensemble/Sensitivity runs
